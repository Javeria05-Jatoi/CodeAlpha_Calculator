# 🧮 Calculator — CodeAlpha Internship Task 2

A clean, modern calculator built as part of my Frontend Development Internship at **CodeAlpha**.

## ✨ Features
- All arithmetic operations: Addition, Subtraction, Multiplication, Division
- Live result preview as you type
- Delete last character with ⌫
- Clear all with AC
- Keyboard support ⌨️
- Responsive & mobile-friendly design
- Smooth hover animations

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (Vanilla)

## 🚀 How to Use
1. Clone or download the repository
2. Open `index.html` in your browser
3. Start calculating!
